Face Expression Recognition using CNN (Real-Time)

This project implements a Convolutional Neural Network (CNN) for facial emotion recognition.
It detects human faces in real-time using OpenCV and classifies emotions such as:
Happy, Sad, Angry, Surprise, Neutral.

The model is trained on grayscale facial datasets and deployed for live webcam inference.

🔹 Features
• Real-time face detection (OpenCV)
• CNN-based emotion classification
• Live webcam inference
• Lightweight and fast

🔹 Tech Stack
• Python
• TensorFlow / Keras (or PyTorch)
• OpenCV
• NumPy
